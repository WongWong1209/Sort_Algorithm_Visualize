# Sort Algorithm Visualize Website
A website designed for visualizing sorting algorithms to make them easier to compare and understand.

[Website Link](https://wongwong1209.github.io/Sort_Algorithm_Visualize/)

## Current Algorithm
* Bubble Sort
* Quick Sort
* Merge Sort
* Counting Sort
* Radix Sort
* Cocktail Sort
